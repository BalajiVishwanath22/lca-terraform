SECTION 1: WHAT IS THE LCA STACK?
================================================================================

The LCA (Live Call Analytics) stack is an AWS solution that:

  1. TRANSCRIBES phone calls in real-time using Amazon Transcribe
  2. ANALYZES call content with AI — detecting sentiment, categories, and issues
  3. ASSISTS agents during live calls with AI-powered suggestions and answers
  4. SUMMARIZES calls automatically using Large Language Models (LLMs)
  5. DISPLAYS everything in a React web UI accessible via CloudFront

Use Case: A call center agent handles a customer call. LCA transcribes it
live, detects the customer's sentiment, surfaces relevant knowledge base
answers for the agent, and generates a summary when the call ends.
